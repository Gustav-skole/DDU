Interaktive ASCII billeder

Tilde symbolet ( ~ ) kan bruges til at lave interaktivt CSS hvor elementer har inflydelse på hinanden. Det har nogle begrænsninger, det virker ikke hvis tilden påvirker noget der komme inden det element som er interaktivt. Du kan implementere det selv vha. følgende mønster

a:state ~ b { style }
